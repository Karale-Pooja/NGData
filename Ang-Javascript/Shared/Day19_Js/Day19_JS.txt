Rest and Spread Operator:

Spread Operator:
	It will help you to combine two arrays or objects together as well as add new members into it.
	
	
Rest Operator:
	It is used to pass params to the function if we dont know exact count of params.
	
Event Bubbling Vs Event Capturing:

	Event Bubbling are from bottom i.e. from child to parent.
	Event Capturing are from top to bottom i.e parent to child.
