Typescript:
TS is completely OOP language.
Typescript is a combination of Old Js + ES6 + Some Additional Features.
Typescript is superset of Javascript.
It was developed by Microsoft Adnderson Hejlberg.
JS is partially OOP language but TS is a fully OOP language.
Browser doesn't understand TS code, so we need to compile down our TS code into JS code so that browswer can run this code.
Converting TS code to JS code is called as Transpilation Process.
TS has 'any' data type which is similar to all the data types.
to run the TS code we need to install typescript.

	npm install typescript -g
